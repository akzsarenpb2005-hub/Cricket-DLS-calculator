# Cricket-DLS-calculator
A Python tool to calculate DLS targets in cricket using Pandas and official ICC resource tables.
This Python application calculates the revised targets for rain-interrupted cricket matches using the **Duckworth-Lewis-Stern (DLS)** method.

## 📊 Features
- Uses **Pandas** to fetch data from the official resource table.
- Implements **ICC Rules** (e.g., minimum 20-over requirement for ODIs).
- Calculates Target Runs and Required Run Rate dynamically.

## 📚 Data & Resources
- **Official Table:** [ICC Standard Resource Table](https://images.icc-cricket.com/image/upload/prd/g9vlypi15msmrfnhucyx.pdf)
- **Logic Reference:** Explained by [Barath Mohan on YouTube](https://youtu.be/J-L586nUFbI)

## 🚀 How to Run
1. Ensure you have Python and Pandas installed: `pip install pandas`
2. Keep `DLS.py` and `DLS.csv` in the same folder.
3. Run the script: `python DLS.py
